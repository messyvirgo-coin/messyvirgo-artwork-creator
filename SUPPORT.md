# Support

This repository contains the Messy Virgo Artwork Creator (`mvac` CLI and local web UI).

## What to expect

- The code here is provided as-is, with best-effort maintenance.
- You are welcome to open issues and pull requests.
  - However, this repo is not a support channel or helpdesk, and maintainers may not respond quickly.
- Maintainers retain editorial control over what gets merged.

## Before opening an issue

- Read the [README](./README.md), especially the "If something goes wrong" section.
- Confirm you are running a supported Python version (3.11+) and have a valid `OPENROUTER_API_KEY`.
- Try the failing command with `--dry-run` to isolate configuration vs. generation issues.

## Where to report bugs

- Use GitHub Issues for reproducible bugs and clear improvement proposals.
- Please include minimal reproduction steps and relevant logs (redacting tokens and sensitive values).

## Security issues

Do not open public issues for suspected vulnerabilities or accidental disclosures.
Instead, report privately to: `contact@messyvirgo.com`

## What not to include

For everyone's safety, do not include secrets, tokens, private links, personal data, or confidential information in issues or pull requests.
